# text_input_game
開発初心者のタイピングゲーム
コードはネット上から拾った

## 使用技術
- HTML
- CSS
- JavaScript

## 機能
- 「ゲームスタート」ボタンをクリックor Enterキーを押してゲームスタート
- 画面中央に表示される文字列をキーボードでタイピング
  - 文字列は5種類しかセットしていない 
- 制限時間は30秒
- 文字列を正しく入力するとスコアが1増えて次の文字列が表示される
- 制限時間が0になるとゲーム終了
  - 画面中央に赤テキストで「ゲーム終了！スコア：｛スコア｝」が表示される
- 「ゲームスタート」ボタンをクリックor Enterキーを押してリスタート

## 起動方法
index.html をブラウザで開く

## 学んだこと
- ネット上から拾ったコードではあるがエラーが出たのでその解決方法を学んだ
- 人の作ったコードを読むこと、コメント挿入、少々長いコードを書くことを学んだ
